Work

Github Link: Children's Home Donations Project

This was an interesting project: integrating SQL into Python for analysis of children's home donation data. I transformed donation data using SQLite3 for seamless database management and Pandas for efficient data manipulation into actionable insights. Visualizations with Seaborn and Matplotlib brought the data to life, pointing out trends and patterns in donation behaviors. This involved deep dives into donation patterns, using SQL queries to output valuable insights in order to optimize fundraising strategies. This project drove home effectively how well the integration of SQL with Python works together and furthermore showed the real power behind data-driven decisions within charity works.

Github Link: Oil Reservoir Productivity Model Project

It was exciting to create a model that would predict oil presence in reservoirs. I started by gathering and cleaning a robust dataset, zeroing in on key features such as depth, porosity, and permeability. With the data prepped at this point, I built a predictive model using advanced machine learning techniques. The fun really picked up when trying out this model through tuning hyperparameters and validation of accuracy through rigorous cross-validation. It was very satisfying to see a model achieve accuracy in predicting the presence of oil, which testified to the precision and beauty of analysis and machine learning.

Github Link: Reservoir Performance Analysis - Hyperbolic Decline Curve and Economic Metrics

Let's dive into the analysis of a shale reservoir with an aggressive decline profile, using hyperbolic decline curves to provide key technical insights such as Estimated Ultimate Recovery (EUR), and business metrics like break-even price projections. We'll also take a fun spin and estimate the probability of maintaining solid production over time using some Monte Carlo magic to uncover the reservoir's true potential!

Web App Link: Analyzing Methane and CO2e Emissions in Oil and Gas: A Well-Based Perspective

At the heart of this project is the idea that every well counts! It is such a thrill to investigate each individual well's emissions using Oil and Gas production data, and seeing how even small tweaks can have a HUGE impact on our planet.
